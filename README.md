##  Descripción del Proyecto
Este proyecto consiste en el desarrollo de una página web simple diseñada para informar sobre las actividades, oferta académica y servicios del **Instituto Politécnico Formosa**. El sitio busca centralizar información relevante como la historia institucional, las tecnicaturas disponibles y los beneficios estudiantiles (comedor, becas y apoyo psicopedagógico) en una interfaz clara y accesible.

---

##  Tecnologías Utilizadas
Para la maquetación y diseño de este sitio se emplearon las siguientes tecnologías:

*   **HTML5:** Para la estructuración semántica del contenido.
*   **CSS3:** Para la personalización de estilos visuales.
*   **Bootstrap 5:** Para lograr un diseño responsivo y utilizar componentes modernos como tarjetas y barras de navegación.
